# Full Stack Trading Platform

A full-stack stock trading platform built using React (Frontend), Spring Boot (Backend), MySQL (Database), and Swagger (API Documentation).

## Features
- User Registration & Authentication
- Wallet Management (Add Funds)
- Buy & Sell Stocks
- Live Stock Price Simulation
- Portfolio Tracking with Profit/Loss
- Transaction History
- RESTful APIs with Swagger Documentation
- Cloud Deployment Ready

## Tech Stack
Frontend: React + Tailwind CSS
Backend: Spring Boot (REST APIs)
Database: MySQL
API Documentation: Swagger (OpenAPI)

## Architecture
- Frontend communicates with Spring Boot REST APIs
- Backend handles trading logic & wallet transactions
- MySQL stores users, portfolio, and transactions
